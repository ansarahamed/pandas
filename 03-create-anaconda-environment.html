1. Open Anaconda Navigator

  • Mac: Finder > Applications > Anaconda Navigator
  • Windows: Start menu > Search for Anaconda Navigator

2. What is an Environment?

  • A sandbox for Python code
  • Lets you install specific versions of Python + libraries
  • Ideal for managing dependencies across multiple projects

3. Base Environment

  • Default environment that comes with Anaconda
  • Includes Python and ~100 libraries like pandas
  • Packages may not be the latest versions

4. Creating a New Environment

  • Click the Environments tab
  • Click Create at the bottom
  • Name the environment (e.g., pandas-playground)
  • Select latest Python version (e.g., 3.11) and click Create

5. Installing Packages

  • Select your new environment
  • Change filter from Installed to All
  • Search and install the following:
    • pandas
    • bottleneck (optimization)
    • numexpr (optimization)
  • Click Apply to install with dependencies

6. Install JupyterLab

  • Go to the Home tab
  • Select your new environment from the dropdown
  • Find the JupyterLab tile and click Install

7. Result

  • You now have an isolated environment with:
    • Latest Python
    • Latest pandas
    • JupyterLab for coding
