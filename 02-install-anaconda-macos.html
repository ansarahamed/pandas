Install Anaconda on macOS

Note: If you're using Windows, feel free to skip this lesson.

Step 1: Search for Anaconda

Open your browser and search for download anaconda on Google. Click the first link to anaconda.com, which takes you to the official download page.

Step 2: Choose the Correct Version

On the download page, locate the Download button. Click the dropdown arrow to select the appropriate installer based on your processor type:

  • Apple Silicon (M1, M2, M3) → Choose the ARM version
  • Intel Mac → Choose the x86_64 version

Step 3: Download the Installer

Choose your version and download the file. It’s large (over 500MB), so it may take a few minutes depending on your internet speed.

Step 4: Install Anaconda

Once downloaded, double-click the installer .pkg file to start.

  • Click "Allow" if prompted
  • Click "Continue" to move through the screens
  • Review the package list (pandas is included!)
  • Accept the license agreement
  • Install for current user (default) or all users
  • Click "Install" and grant access if prompted

Step 5: Wait for Installation

The installation may take a few minutes. Once complete, Anaconda Navigator will launch — you can ignore it for now.

Step 6: Finalize

  • Click "Continue" and then "Close"
  • Move installer to Trash (recommended)

That's It!

You now have the Anaconda distribution installed on your macOS system, including:

  • Python
  • pandas and 100+ data science libraries
  • The conda environment manager

We'll return to Anaconda Navigator in a future lesson when we discuss Windows installation.
